# User Stories

## Freshman

As a freshman,
I want to create an account,
so I can join my school's community.

---

As a freshman,
I want to complete an interest quiz,
so I receive personalized mentor recommendations.

---

As a freshman,
I want to request a mentor,
so I can receive guidance.

---

As a freshman,
I want to message my mentor,
so I can ask questions.

---

As a freshman,
I want to discover clubs,
so I can become involved.

---

As a freshman,
I want to attend events,
so I feel connected.

---

## Link Crew Leader

As a Link Crew leader,
I want to accept mentor requests,
so I can help freshmen.

---

As a Link Crew leader,
I want to answer community questions,
so students receive guidance.

---

## Administrator

As an administrator,
I want to approve mentors,
so only verified leaders mentor students.
